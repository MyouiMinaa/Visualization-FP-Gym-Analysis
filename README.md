# Gym Exercise Trends Visualization

This project is an interactive D3.js dashboard that explores workout data, comparing calories burned and average BPM by workout type and gender.

## Files
- **index.html** — main visualization dashboard  
- **script.js** — logic for filters and D3 rendering  
- **styles.css** — layout and styling  
- **data/gym_members_exercise_tracking.csv** — dataset source  
- **documentation.html** — explanation of design process, rationale, and findings  
- **images/** — sketches and screenshots  


## Dataset Source
[Kaggle - Gym Members Exercise Dataset](https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset)

##  About
Built using **D3.js v7**, HTML, CSS, and JavaScript for my final visualization project.  
This project explores how gender & age make a difference in workout types and how they relate to calories burned and heart rate patterns.
